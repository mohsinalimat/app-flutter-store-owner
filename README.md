# Aniron Dashboard | Flutter App

Specification :

- Flutter 1.17.3
- Dart 2.8.4


## Getting Started

Check Version

- flutter --version

Run Command

- flutter run

SDK build

- flutter build sdk

samples, guidance on mobile development, and a full API reference.
